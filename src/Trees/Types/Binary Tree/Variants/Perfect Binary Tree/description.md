# Perfect Binary Tree

## Overview
A Perfect Binary Tree is the most visually symmetrical tree possible. It is a tree in which **all internal nodes have two children** and **all leaf nodes are at the exact same level**.

## Properties
If a perfect binary tree has a height of `H`:
* Total number of nodes = $2^{H+1} - 1$
* Number of leaf nodes = $2^H$
* It is simultaneously a Full Binary Tree and a Complete Binary Tree.
