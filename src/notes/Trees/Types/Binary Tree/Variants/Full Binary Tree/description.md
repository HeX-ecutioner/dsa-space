# Full Binary Tree

## Overview
A Full Binary Tree (also referred to as a Strictly Binary Tree) is a tree in which **every node has either 0 or 2 children**. 

## Properties
* No node in a Full Binary Tree can have exactly 1 child.
* The number of leaf nodes is always `Number of Internal Nodes + 1`.
* Useful mathematical model for representing mathematical expressions or Huffman coding trees.
